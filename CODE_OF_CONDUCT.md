# Code of Conduct

All contributors are expected to maintain a positive, respectful and inclusive environment.

We follow the principles of respect, collaboration, honesty and transparency.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful of differing viewpoints
- Being open to constructive suggestions
- Demonstrating empathy toward others
- Using inclusive and welcoming language

Examples of unacceptable behavior:

- Harassment of any kind
- Personal attacks
- Disrespectful comments
- Any discriminatory actions

## Enforcement

Project maintainers may remove, edit, or reject comments or contributions that are not aligned with this Code of Conduct.

## Scope

This Code of Conduct applies within all project spaces.
